HOW TO USE PANDORA
====================

판도라 음악 설정 변경 방법을 간략히 소개 하고자 합니다.

첫번째, 가장 간단한 방법은 MOOD: 라고 쓰여있는 판도라 전용 리모컨으로 사용하시면 됩니다. 가능한 설정은, 판도라 전원 끄기, 볼륨 조절, Thumb up, Thumb down 등이 가능합니다. 자세한 조작방법은 아래 첨부파일을 참고하시기 바랍니다.

첨부파일명: remote_how_to.pdf

두번재, Station 을 바꾸거나 Schedule을 변경하려면 반드시 인터넷으로 접속해서 바꿔야 합니다. 핸드폰, 태블렛, 혹은 컴퓨터의 인터넷 브라우져 (Firefox, Chrome, Internet Explorer, Safari, 등) 의 주소 창에 접속 주소를 치시면 됩니다.

접속주소: bit.ly/mgpandora
두개 사진 첨부파일 보시면 Pandora를 인터넷으로 접속 권한, 아이디와 패스워드가 나와있습니다.
