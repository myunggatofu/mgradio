HOW TO USE PANDORA
====================

판도라 음악 설정 변경 방법을 간략히 소개 하고자 합니다.

리모컨 사용
=======
MOOD: 라고 쓰여있는 판도라 전용 리모컨으로 사용하시면 됩니다. 가능한 설정은, 판도라 전원 끄기, 볼륨 조절, Thumb up, Thumb down 등이 가능합니다. 자세한 조작방법은 아래를 참고하시기 바랍니다:

![remote_how_to](https://github.com/myunggatofu/pandora/raw/master/res/remote_how_to.png)

인터넷 브라우저 사용 방법 (핸드폰, 태블렛, 혹은 컴퓨터 사용)
======================================

Station 을 바꾸거나 Schedule을 변경하려면 반드시 인터넷으로 접속해서 바꿔야 합니다.

1. 주소창에 다음을 칩니다: bit.ly/mgradioset
2. user 나 operator 둘 중 하나로 접속합니다. 아래 그림은 user와 operator의 Login 방법과 권한 정보가 나와있습니다:

![userid_user](https://github.com/myunggatofu/pandora/raw/master/res/userid_user.png)
